# 快速开始

欢迎使用 **Nexion** —— 下一代 Web3 SSH 管理器！本文将带你在 5 分钟内完成上手，体验去中心化的服务器管理革命。

## 🚀 什么是 Nexion？

Nexion 是一款创新的 SSH 管理工具，它：
- **用加密钱包替代传统 SSH 密钥**，提升安全性
- **将配置安全存储在区块链上**，实现跨设备同步
- **基于 OKB 的透明定价**，按需付费，节省高达 95% 成本
- **支持多种 Web3 钱包**，包括 OKX Wallet、MetaMask 等

## 📱 第一步：安装 Web3 钱包

### 什么是 Web3 钱包？

Web3 钱包是你的数字身份和资产管理工具，类似于传统的银行卡，但更安全：
- **私钥由你完全控制**，无需信任第三方
- **支持多种加密货币**，包括 OKB、ETH 等
- **一个钱包，多平台使用**，手机、电脑通用

### 初学者钱包设置教程

#### 第一步：创建钱包
1. 下载并安装 OKX 应用
2. 打开应用，选择"创建钱包"
3. 设置安全密码（6-20位字符）
4. 进入备份助记词环节

<img src="https://r2.acexiamo.com/nexion:to-web3-zh.jpg" style={{height: "400px"}} />
<br />

#### 第二步：备份助记词（极其重要！）
1. **记录助记词**：用纸笔抄写12个英文单词
2. **妥善保存**：放在安全的地方，不要截图或云端存储
3. **验证助记词**：按提示顺序点击单词验证

> ⚠️ **重要提醒**：助记词是恢复钱包的唯一方式，丢失助记词 = 丢失资产！

#### 第三步：获取 OKB 代币

##### 对于区块链新手：通过 OKX 交易所购买

1. **法币购买 USDT**
   ```
   OKX首页 → "买币" → 选择 USDT → 输入金额 → 选择支付方式
   ```

<br />

2. **将 USDT 兑换为 OKB**
   ```
   首页 → "交易" → "闪兑" → USDT 换 OKB → 确认兑换
   ```
<img src="https://r2.acexiamo.com/nexion:convert-zh.jpg" style={{height: "400px"}} />

3. **转账到钱包地址**
   ```
   资产 → OKB → 提币 → 输入钱包地址 → 选择 X Layer 网络 → 确认转账
   ```

##### 对于区块链老手：直接转账
如果你已经有 OKB 或其他代币，可以：
- 从其他交易所转账 OKB 到你的钱包地址
- 使用跨链桥将其他链的资产转换为 X Layer 上的 OKB
- 通过 DEX 交换其他代币为 OKB

#### 第四步：切换到 X Layer 网络

Nexion 运行在 **X Layer** 区块链上，需要切换网络：

1. **打开钱包设置**
   ```
   钱包首页 → 右上角网络名称 → 添加网络
   ```

2. **切换到 X Layer**
   - 添加成功后，选择 X Layer 作为当前网络
   - 确保你的 OKB 在 X Layer 网络上

### 为什么 Nexion 更安全？

**传统 SSH 管理的问题：**
- 私钥文件容易丢失或被盗
- 配置存储在本地，难以同步
- 密码认证容易被暴力破解
- 无法追踪访问历史

**Nexion 的 Web3 解决方案：**
- **钱包签名认证**：使用加密学级别的数字签名，比传统密码安全千倍
- **区块链存储**：配置加密存储在去中心化网络，永不丢失
- **身份去中心化**：你的钱包地址就是身份，无需注册账号
- **操作可追溯**：所有操作记录在区块链上，完整审计历史

## 🔗 第三步：连接钱包

### 连接流程

首次启动 Nexion 时：

1. **点击"连接钱包"**
   - 应用会自动检测可用的钱包
   - 推荐使用 OKX Wallet

2. **选择钱包类型**
   ```
   OKX Wallet (推荐) - 专为 OKB 生态优化
   MetaMask - 广泛使用的以太坊钱包
   WalletConnect - 支持多种移动钱包
   ```

3. **授权连接**
   - 钱包会弹出连接请求
   - 点击"连接"授权应用访问

4. **身份验证签名**
   - 应用要求签名验证身份
   - 签名是免费的，不消耗代币
   - 点击"签名"完成身份绑定

### 我们如何使用你的钱包？

**Nexion 基于钱包的实现方式：**

1. **身份认证**
   ```
   钱包签名 = 身份验证
   钱包地址 = 唯一用户ID，无需注册账号
   ```

2. **配置管理**
   ```
   支持密码和私钥两种SSH认证方式
   添加/修改配置时需要钱包签名确认
   ```

3. **数据加密存储**
   ```
   配置信息 → 钱包签名加密 → 上传区块链
   获取配置 → 基于签名解密 → 显示给用户
   只有你的钱包能解密自己的配置
   ```

4. **费用结构**
   ```
   收费操作: 注册、新增配置、修改配置 (一次性OKB费用)
   免费操作: 查询配置、SSH连接、FTP传输 (无任何费用)
   无订阅模式，按需付费
   ```

**安全保障：**
- 🔐 **钱包签名加密**：配置使用签名结果加密，绝对安全
- 🛡️ **私钥不离开钱包**：Nexion 无法访问你的私钥  
- 📝 **操作可审计**：所有价值交互记录在区块链上
- ⚡ **去中心化存储**：配置存储在区块链，永不丢失
