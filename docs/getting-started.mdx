# 快速开始

欢迎使用 **Nexion** —— 下一代 Web3 SSH 管理器！本文将带你在 5 分钟内完成上手，体验去中心化的服务器管理革命。

## 🚀 什么是 Nexion？

Nexion 是一款创新的 SSH 管理工具，它：
- **用加密钱包替代传统 SSH 密钥**，提升安全性
- **将配置安全存储在区块链上**，实现跨设备同步
- **基于 OKB 的透明定价**，按需付费，节省高达 95% 成本
- **支持多种 Web3 钱包**，包括 OKX Wallet、MetaMask 等

## 📥 安装与设置

### 1. 下载应用

访问我们的 [下载页面](#download)，选择适合你操作系统的版本：
- **Windows** - `.exe` 安装包
- **macOS** - `.dmg` 安装包  
- **Linux** - `.AppImage` 或 `.deb` 包

### 2. 连接 Web3 钱包

首次启动时：
1. 点击"连接钱包"按钮
2. 选择你的钱包类型（推荐 **OKX Wallet**）
3. 授权连接并签名验证身份
4. 完成！你的身份已绑定到区块链

## 🖥️ 添加第一台服务器

### 基本配置

1. 在主界面点击 **"新增服务器"**
2. 填写服务器信息：
   ```
   服务器名称: 我的云服务器
   主机地址: 192.168.1.100
   端口: 22
   用户名: root
   ```
3. 点击"保存并上链"

### 高级配置

对于更复杂的场景，你还可以配置：
- **私钥文件路径** - 使用现有的 SSH 私钥
- **跳板机设置** - 通过跳板机连接内网服务器
- **环境变量** - 预设连接后的环境变量
- **自动执行命令** - 连接后自动运行的脚本

## 🔗 发起 SSH 连接

配置完成后，连接服务器只需：

1. **选择服务器** - 在列表中找到目标服务器
2. **一键连接** - 点击"连接"按钮
3. **自动验证** - 系统从区块链加载配置并建立连接
4. **开始工作** - SSH 终端自动打开，开始你的工作

## 💰 计费说明

Nexion 采用**按需付费**模式：
- **存储费用**: 0.001 OKB/月/服务器配置
- **连接费用**: 0.0001 OKB/次连接
- **数据同步**: 免费（基于 X Layer 网络）

## 🔧 常见问题

### 钱包连接失败？
- 确保钱包已解锁
- 检查网络连接
- 尝试刷新页面后重新连接

### 服务器配置丢失？
- 检查钱包地址是否正确
- 确认网络连接正常
- 配置数据存储在区块链上，永不丢失

### 连接速度慢？
- 第一次连接需要从区块链同步数据
- 后续连接会使用本地缓存，速度更快
