# 安装指南

Nexion 是一款基于 Web3 技术的现代化 SSH 管理工具，支持多平台部署。

## 📥 下载

### 当前可用版本

**macOS** - 已发布
- 支持 Intel 和 Apple Silicon 芯片
- 最低版本要求：macOS 10.15+

**Windows / Linux** - 敬请期待
- 正在积极开发中
- 计划近期发布

### 下载地址

访问我们的 [GitHub Releases](https://github.com/AceXiamo/Nexion/releases) 页面获取最新版本。

## 🍎 macOS 安装

1. **下载 DMG 文件**
   - 从 GitHub Releases 下载 `Nexion-x.x.x.dmg`

2. **安装应用**
   - 双击 DMG 文件
   - 将 Nexion 拖拽到 Applications 文件夹

3. **首次启动**
   - 从 Applications 文件夹启动 Nexion
   - 如遇安全提示，前往 **系统偏好设置 → 安全性与隐私** 点击"仍要打开"

## 🔧 初始设置

首次启动时需要：

1. **连接 Web3 钱包** - 推荐使用 OKX Wallet
2. **选择网络** - 默认使用 X Layer 主网
3. **完成身份验证** - 通过钱包签名验证身份

## 🔄 自动更新

Nexion 内置自动更新功能：
- 启动时自动检查新版本
- 一键升级到最新版本
- 配置数据自动迁移
